# Ark Nova Open-Source Engine Candidates

Checked on 2026-03-04.

## Selected (integrated)

1. `jbargu/TabletopGames`  
   URL: https://github.com/jbargu/TabletopGames  
   License: MIT  
   Notes: Contains `games/arknova` source and tests. Current implementation is partial but usable as baseline.

## Why `TabletopGames`

- Clear game-state/action model for AI research.
- MIT license is compatible with integration.
- Has concrete Ark Nova map/build logic and tests we can reuse.
